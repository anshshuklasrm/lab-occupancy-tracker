Lab Occupancy Display

Real-time monitoring of lab spots across Norman, Satoshi, Pausch, and McCarthy

How This System Works

Spot status is updated through Arduino devices located in each lab.

When a person enters/exits, they press the corresponding button on the device.

This display reflects the current state from the backend - use the refresh button to get the latest data.
